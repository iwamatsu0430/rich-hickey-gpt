## Get Started

```sh
$ npm install
$ export OPENAI_API_KEY=YOUR_API_KEY # recommend direnv
```

Run

```sh
$ npm run answer "What is Simple?"
```

### Optional

Scrape and refresh data.db

```sh
$ npm run scrape
```

Embedding

```sh
$ npm run embed
```
